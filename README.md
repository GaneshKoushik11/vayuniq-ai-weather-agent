# 🌤️ AI-Powered Weather Agent

An intelligent, conversational weather assistant built using a modern full-stack architecture. By combining the speed of **Groq** with LLM **tool calling**, this agent moves beyond static chat responses to autonomously fetch real-time meteorological data and interpret it contextually for the user.

### 🛠️ Key Features
* **Agentic Architecture:** Uses Groq's LLM capabilities to understand intent, reason, and execute function calls dynamically.
* **Real-Time Tool Calling:** Seamlessly bridges the gap between natural language and live weather APIs using structured Node.js tool integration.
* **Interactive UI:** A highly responsive React-based chat interface designed for fluid, human-like conversations.
* **Context-Aware Insights:** Goes beyond raw numbers to provide actionable recommendations (e.g., clothing suggestions, travel advice) based on live conditions.

### 💻 Tech Stack
* **Frontend:** React, TypeScript (optional), CSS/Component Library
* **Backend:** Node.js, Express
* **AI Orchestration:** Groq SDK, Function/Tool Calling
